# Step to run client

1. go inside client dirctory
2. run "npm install"
3. run "npm run dev"

client will start at localhost:8000

# Step to run server

1. go inside server dirctory
2. run "npm install"
3. run "npm run start"

server will start listening at localhost:5000

Prerequisit: MySQL and REDIS needs to be install on host machine
